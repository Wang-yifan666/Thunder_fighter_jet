# 雷霆战机 v1.0

一个基于 C++ 和 FTXUI 的终端版雷霆战机小游戏，实现了玩家移动、生命显示、FPS、计时和等级系统。

---

## 项目概述

- 使用 **FTXUI** 构建终端 UI 界面。
- 玩家使用 **WASD** 控制飞机上下左右移动。
- 显示 **FPS** 和游戏已运行时间。
- 底部显示 **剩余生命**。
- 自动根据游戏时间更新 **等级**。
- 支持退出游戏（按 `Q` 或 `Esc` 键）。
